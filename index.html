Decentralized settlement

DCRS is a digital currency that can be settled by everyone in a currency, and can be used to realize a bank that can be settled anywhere in the world.

Untraceable payments

DCRS provides users with a completely anonymous payment scheme. DCRS implements the ring signature technology which allows you to sign a message on behalf of a group. The signature only proves the message was created by someone from the group, but all the possible signers are indistinguishable from each other.

Secure and private

DCRS is based on the CryptoNote implementation, which provides complete anonymous payments, millisecond determinations, undetectable payments, dual spending certifications, and future features.
